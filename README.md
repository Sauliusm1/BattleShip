# BattleShip

Optas internship assginment

## Prerequisites
Nodejs, npm
## Project Setup

``` sh
git clone https://github.com/Sauliusm1/BattleShip.git
```

### Back end setup
```sh
cd BackEnd
npm install
```

#### Run with
```sh
node index.js
```

### Front end set up

```sh
cd FrontEnd
npm install
```

#### Run with
```sh
npm run dev
```
